# Aby Alex Jays | Personal Portfolio

A cyber forensics and pentesting-themed personal portfolio website. This project serves as a digital resume and project showcase, designed with a deep-space tech aesthetic, featuring an terminal interface, dynamic network node animations, and a fully responsive layout.

## ✨ Key Features
* **Terminal Interfaces:** The Hero and Contact sections mimic a Linux terminal environment for a native cybersecurity feel.
* **Network Node Animation:** A custom, lightweight HTML5 `<canvas>` background animation featuring drifting data nodes and connection paths.
* **Project Hover UI:** Sleek, frosted-glass overlay effects that reveal repository and live-site links dynamically.
* **Fully Responsive:** Fluid layouts using CSS Flexbox and Grid, ensuring the site looks exceptional on large desktop monitors, tablets, and mobile devices.
* **Custom Theming:** Deep amethyst purple and cyber-blue gradients with strict, minimalist typography.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom variables, CSS animations, Flexbox, Grid, and glassmorphism styling.
* **JavaScript (Vanilla):** DOM manipulation, smooth scrolling, and Canvas API mechanics.

## 📂 Sections Included
1.  **Home:** Terminal intro, core mission statement, and quick links.
2.  **Education:** Academic background and current B.Sc progress.
3.  **Skills:** Categorized expertise spanning Artifact Analysis, nmap, React.js, and Python.
4.  **Projects:** Featured repositories including AstroBio-Explorer, Fire Store, and CipherCheck.
5.  **Contact:** A functional, terminal-styled connection block linking to Email, LinkedIn, and GitHub.
